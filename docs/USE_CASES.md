# 11. Програма лояльності

Після завершення платного візиту

```
loyaltyVisits +1

↓

totalVisits +1

↓

Перевірити

loyaltyVisits == freeVisitEvery
```

Якщо так

```
freeVisitBalance +1

↓

loyaltyVisits = 0
```

Якщо використовується безкоштовне відвідування

```
freeVisitBalance -1

↓

freeVisitsUsed +1

↓

totalVisits +1
```

# 19. Правила доступу

## PARENT

✔ бачить лише своїх дітей

✔ бачить лише свої відвідування

✔ може створювати нові картки дітей

✔ може починати візити

✖ не бачить інших користувачів

✖ не бачить фінансової статистики

---

## ADMIN

✔ бачить усі картки дітей

✔ підтверджує початок візитів

✔ завершує візити

✔ переглядає активні візити

✔ переглядає статистику за день

✖ не бачить доходів

✖ не може змінювати налаштування

---

## SUPER_ADMIN

Успадковує всі права ADMIN.

Додатково

✔ доходи

✔ аналітика

✔ керування адміністраторами

✔ налаштування системи

✔ повний доступ до всіх даних.
